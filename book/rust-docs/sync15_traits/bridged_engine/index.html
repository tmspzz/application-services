IncomingEnvelope

An envelope for an incoming item, passed to BridgedEngine::store_incoming. Envelopes are a halfway point between BSOs, the format used for all items on the Sync server, and records, which are specific to each engine.

OutgoingEnvelope

An envelope for an outgoing item, returned from BridgedEngine::apply. This is conceptually identical to IncomingEnvelope, but omits fields that are only set by the server, like modified.
